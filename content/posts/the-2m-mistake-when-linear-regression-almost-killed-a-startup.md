**Q:** You're building a customer churn prediction model. Given a dataset with customer features (age, usage frequency, subscription type), how would you determine whether to use linear regression or logistic regression?

**A:** Use logistic regression for binary churn prediction (yes/no), linear regression for continuous values like predicted churn time or revenue loss.
