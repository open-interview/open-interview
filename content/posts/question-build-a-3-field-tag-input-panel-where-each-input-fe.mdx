> **Understanding **Question**: Build a 3-field tag input panel where each input fetches suggestions from `/api/suggest?q=TERM` as the user types. Debounce per field, cancel previous in-flight requests on new input, and ensure only the latest response updates the UI. Enforce a global limit of 4 concurrent fetches. Use closures to encapsulate per-field state and Promises to handle results?**

## Overview

Leverage per-field closures to store a debounce timer and an AbortController; on input, reset the timer and, when it fires, start a fetch with its own controller. Track a global in-flight counter to cap concurrent requests at 4, decrementing it on completion or cancellation.
