> **Understanding Channel: backend/authentication; Difficulty: advanced; Topics: jwt, oauth2, oidc, saml. You are building a high-security API gateway that must support both user-attested AuthN (OIDC/SAML) and device-attested AuthN via FIDO2 attestation. The gateway issues an internal JWT containing tenant, user_id, device_id, and a per-tenant nonce. Outline end-to-end flow and provide concrete snippets: 1) OpenAPI security for the internal JWT, 2) a FIDO2 attestation verifier, 3) middleware that binds device and user, enforces audience, and prevents replay?**

## Overview

## End-to-End Authentication Flow

The authentication flow proceeds as follows:

1. **User Authentication**: Validate the OIDC/SAML assertion against tenant-specific JWKS, enforcing correct `audience` and `issuer` claims.
2. **Device Attestation**: Perform FIDO2 attestation verification to bind a `device_id` to the user within the tenant context.
3. **Token Issuance**: Generate an internal JWT containing:
   - `sub`: user identifier
   - `tenant`: tenant identifier
   - `device_id`: FIDO2-attested device identifier
   - `aud`: internal API gateway audience
   - `nonce`: per-tenant nonce for replay protection
   - `iat` and `exp`: issuance and expiration timestamps

## 2. FIDO2 Attestation Verifier

```javascript
async function verifyFIDO2Attestation(attestation, userId, tenant) {
  const { attestationObject, clientDataJSON } = attestation;
  
  const clientData = JSON.parse(atob(clientDataJSON));
  
  if (clientData.challenge !== expectedChallenge) {
    throw new Error('Invalid attestation challenge');
  }
  
  if (clientData.origin !== expectedOrigin) {
    throw new Error('Invalid attestation origin');
  }
  
  const attestationStruct = cbor.decode(attestationObject);
  const { authData } = attestationStruct;
  
  const deviceId = deriveDeviceId(authData.attestedCredentialData.credentialId);
  
  await bindDeviceToUser(deviceId, userId, tenant);
  
  return { deviceId, verified: true };
}
```

## 3. JWT Validation Middleware

```javascript
function createAuthMiddleware(tenantConfig) {
  return async (req, res, next) => {
    const token = extractBearerToken(req);
    const claims = await verifyJWT(token, tenantConfig.jwks);
    
    if (!validateAudience(claims.aud, tenantConfig.audience)) {
      return res.status(403).json({ error: 'Invalid audience' });
    }
    
    if (await isNonceReused(claims.nonce, claims.tenant)) {
      return res.status(401).json({ error: 'Replay attempt detected' });
    }
    
    if (!await validateDeviceBinding(claims.device_id, claims.sub, claims.tenant)) {
      return res.status(403).json({ error: 'Device binding invalid' });
    }
    
    req.auth = {
      userId: claims.sub,
      tenant: claims.tenant,
      deviceId: claims.device_id
    };
    
    next();
  };
}
```
