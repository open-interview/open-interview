**Q:** What is the difference between CPU profiling and memory profiling, and when would you use a flame graph?

**A:** CPU profiling measures execution time spent in functions, while memory profiling tracks memory allocation patterns and usage. Flame graphs are used to visualize CPU bottlenecks and identify performance hotspots.
